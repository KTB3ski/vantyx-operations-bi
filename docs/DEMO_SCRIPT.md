# VANTYX Demo Script

Use this as a short 60-120 second walkthrough script for a portfolio video or live demo.

## Setup

Open the app in browser demo mode or desktop mode with sample data loaded.

## Script

1. "VANTYX is a local-first operations BI tool for hotel GOP recovery workflows."
2. "The goal is to move from raw operating numbers to a clear read: current position, forecast, gap, and recovery actions."
3. Show the property/date setup and explain that demo data is editable sample data.
4. Open the results view and point out:
   - Current position
   - Forecasted position
   - Recovery revenue
   - Confidence/readout
5. Show the action plan and explain how the tool turns the gap into operator-facing next steps.
6. Show CSV export or print/PDF-style reporting.
7. Close with: "This is an active prototype. The public repo contains source and sample/demo assets only, no client data or private credentials."

## What To Avoid Saying

- Do not claim this is deployed enterprise software.
- Do not imply it connects to live hotel systems today.
- Do not claim AI makes financial decisions.
- Do not show private customer, hotel, credential, or production data.
